# Excel 品牌规范提示词 v1.0

> 适用场景：生成 Excel 报表、数据导出文件时使用此提示词，让 AI 生成的 Excel 文件符合 Comlan 品牌规范。
> 直接复制全文粘贴至对话框即可。

---

## 品牌色规范（强制）

本项目的 Excel 文档仅使用以下颜色，禁止随意引入其他色相：

| 角色 | 色值 | 名称 | Excel 用途 |
|------|-------|------|----------|
| 主色 | `#002FA7` | 克莱因蓝 | 表头背景、Sheet 标签颜色、品牌标识单元格 |
| 辅色 | `#00C3FF` | 浅蓝 | 分类列背景、图表数据系列 1、次级强调 |
| 深色变体 | `#001D70` | 深克莱因蓝 | 重点数据单元格、图表数据系列 2 |
| 浅色变体 | `#CAD5F6` | 浅蓝紫 | 交替行背景色（奇数行） |

**RAG 语义色（Red-Amber-Green，红-琥珀-绿，仅用于状态指示）：**

按 R→A→G 顺序排列，严格对应状态严重程度：

| 状态 | 色值 | Excel 用途 |
|------|-------|-----------|
| 失败/异常/告警（Red） | `#ef4444` | 条件格式：数值异常（红底或红字） |
| 警告/临界（Amber） | `#f59e0b` | 条件格式：数值接近阈值（黄底） |
| 成功/正常（Green） | `#10b981` | 条件格式：数值达标（绿底或绿字） |

> ⚠️ **设计纪律**：Excel 整体颜色不超过 3 种（品牌色 + 中性灰）。RAG 色仅用于条件格式，不用于装饰性单元格。

## 中性色板（推荐使用）

| 用途 | 色值 | Excel 位置 |
|------|-------|-----------|
| 纯白背景 | `#FFFFFF` | 数据区域背景 |
| 页面背景灰 | `#F0F4F8` | 说明区域背景 |
| 主文字 | `#1E293B` | 所有数据文字 |
| 次要文字 | `#64748B` | 注释、单位说明 |
| 占位文字 | `#94A3B8` | 模板占位符 |
| 边框 | `#E2E8F0` | 所有单元格边框 |

## Excel 表格样式规范

### 表头行（第 1 行）

| 属性 | 规范 |
|------|------|
| 背景色 | `#002FA7`（克莱因蓝） |
| 文字色 | `#FFFFFF`（白色） |
| 字体 | Microsoft YaHei，11pt，**粗体** |
| 对齐 | 水平居中，垂直居中 |
| 边框 | 底部 `##E2E8F0` 1px 实线 |
| 冻结 | 必须冻结首行（`Freeze Panes`） |

### 数据行

| 属性 | 规范 |
|------|------|
| 背景色 | 白色（偶数行）/ `#F8FAFC`（奇数行，极浅灰） |
| 文字色 | `#1E293B` |
| 字体 | Microsoft YaHei，10pt，常规 |
| 数字格式 | 千位分隔符（如 `1,234`）、保留两位小数（金额类） |
| 边框 | 全部 `#E2E8F0` 0.5pt 实线 |
| 交替行 | 使用"表格样式"或条件格式实现浅灰交替 |

### 汇总行（末行）

| 属性 | 规范 |
|------|------|
| 背景色 | `#CAD5F6`（浅蓝紫） |
| 文字色 | `#002FA7`（克莱因蓝） |
| 字体 | Microsoft YaHei，10pt，**粗体** |
| 顶部边框 | `#002FA7` 2pt 双线 |
| 公式 | `SUBTOTAL` 或 `SUM`，根据列类型选择 |

## Excel 图表配色方案

### 柱状图 / 条形图

| 数据系列 | 填充色 | 边框 |
|----------|-------|------|
| 系列 1（主指标） | `#002FA7` | 无 |
| 系列 2（次指标） | `#00C3FF` | 无 |
| 系列 3（第三指标） | `#CAD5F6` | 无 |
| 峰值高亮柱 | `#001D70` | 无 |
| 网格线 | `#E2E8F0` | — |
| 坐标轴文字 | `#64748B` | — |

### 折线图

| 数据系列 | 线条色 | 宽度 | 标记点 |
|----------|-------|------|--------|
| 系列 1 | `#002FA7` | 2.25pt | 圆形，5pt，`#FFFFFF` 填充 |
| 系列 2 | `#00C3FF` | 2.25pt | 圆形，5pt，`#FFFFFF` 填充 |
| 趋势线 | `#001D70` | 1.5pt，虚线 | 无 |

### 饼图 / 环形图

使用品牌色渐变，禁止用 Excel 默认彩虹色盘：

```
配色顺序（按数据占比从大到小）：
1. #002FA7（克莱因蓝）
2. #001D70（深蓝）
3. #00C3FF（浅蓝）
4. #66D9FF（更浅蓝）
5. #CAD5F6（浅蓝紫）
```

**饼图标签规范：**
- 标签位置：数据标签在外（彩色填充饼图用"最佳匹配"）
- 文字颜色：与扇区同色 或 白色（深色扇区）
- 字体：Microsoft YaHei，9pt

## 条件格式规范

### 数值达标/异常

| 规则类型 | 条件 | 格式 |
|---------|------|------|
| 高于目标 | 值 ≥ 目标值 | 字体 `#10b981`（绿），加粗 |
| 低于阈值 | 值 < 阈值 | 字体 `#ef4444`（红），加粗 |
| 接近阈值 | 阈值 × 0.9 ≤ 值 < 阈值 | 字体 `#f59e0b`（琥珀），加粗 |
| Top 3 | 前 3 名 | 单元格背景 `#CAD5F6`，字体 `#002FA7` 加粗 |
| 重复值 | 重复 | 字体 `#ef4444` |

### 数据条（KPI 类）

- 颜色：`#002FA7`（实心填充，不渐变）
- 边框：无
- 坐标轴：自动（最小值 0）
- 负值时：禁用（数据条仅用于正数）

## Sheet 组织结构规范

### 多 Sheet 工作簿

| Sheet 位置 | 命名规范 | 颜色标签 |
|-----------|-----------|----------|
| 第 1 个 | `汇总` 或 `总览` | `#002FA7`（蓝） |
| 第 2 个 | `明细数据` 或具体业务名 | `#00C3FF`（浅蓝） |
| 第 3 个 | `图表分析` | 无（默认灰） |
| 最后 1 个 | `说明` 或 `字段定义` | 无（默认灰） |

### 打印设置

| 属性 | 规范 |
|------|------|
| 纸张方向 | 列数 ≤ 6 用纵向，> 6 用横向 |
| 页边距 | 窄边距（上 1.9cm，其余 1.3cm） |
| 页眉 | 左：Comlan 昆仑联通；右：打印日期 |
| 页脚 | 居中：第 &[Page] 页 / 共 &[Pages] 页 |
| 重复标题行 | 必须设置（每页重复表头） |
| 网格线 | 打印时**不显示** |

## 设计审查清单

生成 Excel 文件后，必须核对：

- [ ] 表头背景是否为 `#002FA7`，文字是否为白色粗体
- [ ] 是否有多余的颜色引入（红/橙/紫/粉等，RAG 除外）
- [ ] 图表配色是否使用品牌色板，而非 Excel 默认彩虹色
- [ ] 条件格式是否仅使用 RAG 语义色
- [ ] 数字格式是否统一（千位分隔、小数位数）
- [ ] 是否冻结了首行（表头）
- [ ] 打印设置是否隐藏网格线
- [ ] Sheet 标签颜色是否符合规范

## 提示词使用方式

**下次需要生成 Excel 文件时，将以下内容粘贴给 AI：**

```
请按照以下 Comlan 品牌规范生成 Excel 文件：

【品牌色】
- 主色：#002FA7（克莱因蓝）— 表头背景、主图表系列
- 辅色：#00C3FF（浅蓝）— 次图表系列、分类列背景
- 深色变体：#001D70 — 重点数据、图表系列 2
- RAG 语义色（Red-Amber-Green，红-琥珀-绿）：`#ef4444`（失败/告警）→ `#f59e0b`（警告）→ `#10b981`（成功），仅用于条件格式

【表格样式】
- 表头：#002FA7 背景、白色粗体、冻结首行
- 数据行：白色/#F8FAFC 交替、#1E293B 文字、#E2E8F0 边框
- 汇总行：#CAD5F6 背景、#002FA7 粗体、顶部 2pt 双线

【图表配色】
- 柱状图系列 1：#002FA7，系列 2：#00C3FF
- 饼图配色顺序：#002FA7 → #001D70 → #00C3FF → #66D9FF → #CAD5F6
- 禁止使用 Excel 默认彩虹色盘

【打印设置】
- 隐藏网格线、重复标题行、页眉左：Comlan 昆仑联通

请严格遵守以上规范生成 Excel 文件。
```
