# Comlan 昆仑联通 - PPT 文档品牌规范提示词 v1.0

> 适用于所有需要生成 Microsoft PowerPoint (PPT/PPTX) 文档的场景。  
> 直接复制本文档内容粘贴给 AI，即可生成符合 Comlan 品牌规范的 PPT 演示文稿。

## 🎨 品牌色规范（强制）

### 基础品牌色

| 角色 | 色值 | 名称 | PPT 用途 |
|------|-------|------|-----------|
| 主色 | `#002FA7` | 克莱因蓝 | 标题、品牌标识、图表系列 1、重点数据 |
| 辅色 | `#00C3FF` | 浅蓝 | 次级标题、图表系列 2、强调色、超链接 |
| 深色变体 | `#001D70` | 深克莱因蓝 | 深色背景区域、图表系列 2（深色版） |
| 浅色变体 | `#CAD5F6` | 浅蓝紫 | 表格交替行、背景装饰、次要元素 |

### RAG 语义色（Red-Amber-Green，红-琥珀-绿，仅用于状态指示）

按 R→A→G 顺序排列，严格对应状态严重程度：

| 状态 | 色值 | PPT 用途 |
|------|-------|-----------|
| 失败/异常/告警（Red） | `#ef4444` | 条件格式：数值异常（红字或红底） |
| 警告/临界（Amber） | `#f59e0b` | 条件格式：数值接近阈值（黄底） |
| 成功/正常（Green） | `#10b981` | 条件格式：数值达标（绿字或绿底） |

> ⚠️ **设计纪律**：PPT 文档整体颜色不超过 3 种（品牌色 + 中性灰）。RAG 色仅用于状态，不用于装饰性元素。

### 中性色板

| 用途 | 色值 | PPT 位置 |
|------|-------|-----------|
| 纯白背景 | `#FFFFFF` | 页面背景（主色调） |
| 深蓝背景 | `#001D70` | 封面页、章节页背景 |
| 主文字 | `#1E293B` | 所有正文文字（浅色背景上） |
| 浅色文字 | `#FFFFFF` | 深色背景上的文字 |
| 次要文字 | `#64748B` | 注释、图注 |
| 边框 | `#E2E8F0` | 表格边框、分隔线 |

## 📊 PPT 幻灯片布局规范

### 封面页（第 1 页）

```
元素              规范
----------------   -----------------------------------------
背景              深色 #001D70 或渐变（#002FA7 → #001D70）
公司 Logo         左上角或居中，白色，建议尺寸 120×40px
文档标题          白色，Microsoft YaHei，32pt，加粗
副标题            浅蓝 #00C3FF，Microsoft YaHei，18pt
日期              白色，Microsoft YaHei，12pt，透明度 70%
装饰元素          浅蓝紫 #CAD5F6 几何图形，置于背景层
```

### 标准内容页

```
元素              规范
----------------   -----------------------------------------
背景              白色 #FFFFFF 或浅灰 #F0F4F8（可选）
幻灯片标题        克莱因蓝 #002FA7，Microsoft YaHei，24pt，加粗
标题下装饰线      3pt 高度，克莱因蓝 #002FA7 填充，居中
正文文字          深灰 #1E293B，Microsoft YaHei，14pt
项目符号          ● 克莱因蓝 #002FA7
页码              深灰 #64748B，Microsoft YaHei，9pt，右下角
```

### 章节分隔页

```
元素              规范
----------------   -----------------------------------------
背景              克莱因蓝 #002FA7 或渐变（#002FA7 → #00C3FF）
章节标题          白色，Microsoft YaHei，36pt，加粗
章节副标题        浅蓝 #00C3FF，Microsoft YaHei，16pt
页码              白色，Microsoft YaHei，9pt，右下角
```

### 饼图 / 环形图

配色顺序（按数据占比从大到小）：
1. `#002FA7`（克莱因蓝）
2. `#001D70`（深蓝）
3. `#00C3FF`（浅蓝）
4. `#66D9FF`（更浅蓝）

**饼图标签规范：**
- 标签位置：数据标签在外（彩色填充饼图用"最佳匹配"）
- 文字颜色：与扇区同色 或 白色（深色扇区）
- 字体：Microsoft YaHei，9pt

### 图标使用建议

- 优先使用线条风格图标（Outline Icons）
- 图标颜色：`- #002FA7`（主图标）或 `#00C3FF`（次要图标）
- 图标尺寸：统一为 32×32px 或 48×48px
- 图标风格：扁平化、简洁、现代

## 🎯 设计审查清单

生成 PPT 文档后，必须核对：

- [ ] 封面页是否使用品牌色背景（深蓝 #001D70 或渐变）
- [ ] 幻灯片标题是否使用品牌色（`#002FA7`）
- [ ] 是否有多余的颜色引入（红/橙/紫/粉等，RAG 除外）
- [ ] 表格表头是否为 `#002FA7` 背景 + 白色文字
- [ ] 图表配色是否使用品牌色板，而非 PPT 默认配色
- [ ] 页眉页脚是否包含 Comlan 品牌标识
- [ ] 字体是否统一使用 Microsoft YaHei
- [ ] 数字格式是否统一（千位分隔、小数位数）
- [ ] 是否有章节分隔页（如内容较多）
