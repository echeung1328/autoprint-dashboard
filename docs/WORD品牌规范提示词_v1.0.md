# Comlan 昆仑联通 - WORD 文档品牌规范提示词 v1.0

> 适用于所有需要生成 Microsoft Word (DOCX) 文档的场景。  
> 直接复制本文档内容粘贴给 AI，即可生成符合 Comlan 品牌规范的 Word 文档。

## 🎨 品牌色规范（强制）

### 基础品牌色

| 角色 | 色值 | 名称 | Word 用途 |
|------|-------|------|-----------|
| 主色 | `#002FA7` | 克莱因蓝 | 标题、表头背景、品牌标识、图表系列 1 |
| 辅色 | `#00C3FF` | 浅蓝 | 次级标题、图表系列 2、强调色 |
| 深色变体 | `#001D70` | 深克莱因蓝 | 重点数据、图表系列 2（深色版） |
| 浅色变体 | `#CAD5F6` | 浅蓝紫 | 表格交替行、汇总行背景 |

### RAG 语义色（Red-Amber-Green，红-琥珀-绿，仅用于状态指示）

按 R→A→G 顺序排列，严格对应状态严重程度：

| 状态 | 色值 | Word 用途 |
|------|-------|-----------|
| 失败/异常/告警（Red） | `#ef4444` | 条件格式：数值异常（红字或红底） |
| 警告/临界（Amber） | `#f59e0b` | 条件格式：数值接近阈值（黄底） |
| 成功/正常（Green） | `#10b981` | 条件格式：数值达标（绿字或绿底） |

> ⚠️ **设计纪律**：Word 文档整体颜色不超过 3 种（品牌色 + 中性灰）。RAG 色仅用于状态，不用于装饰性元素。

### 中性色板

| 用途 | 色值 | Word 位置 |
|------|-------|-----------|
| 纯白背景 | `#FFFFFF` | 页面背景 |
| 页面背景灰 | `#F0F4F8` | 说明文本框背景 |
| 主文字 | `#1E293B` | 所有正文文字 |
| 次要文字 | `#64748B` | 注释、图注 |
| 边框 | `#E2E8F0` | 表格边框、分隔线 |

## 📄 Word 文档样式规范

### 封面页

```
元素              规范
----------------   -----------------------------------------
公司 Logo         左上角或居中，Comlan 品牌标识
文档标题          克莱因蓝 #002FA7，Microsoft YaHei，24pt，加粗
副标题            浅蓝 #00C3FF，Microsoft YaHei，16pt
日期              深灰 #64748B，Microsoft YaHei，11pt
顶部装饰条        4pt 高度，克莱因蓝 #002FA7 填充
```

### 标题样式

| 级别 | 字体 | 字号 | 颜色 | 间距（前/后） |
|------|------|------|------|--------------|
| 标题 1 | Microsoft YaHei | 18pt | `#002FA7` | 24pt / 6pt |
| 标题 2 | Microsoft YaHei | 14pt | `#002FA7` | 18pt / 4pt |
| 标题 3 | Microsoft YaHei | 12pt | `#00C3FF` | 12pt / 3pt |
| 正文 | Microsoft YaHei | 10.5pt | `#1E293B` | 0pt / 6pt |
| 图注 | Microsoft YaHei | 9pt | `#64748B` | 6pt / 6pt |

### 表格样式

#### 表头行

| 属性 | 规范 |
|------|------|
| 背景色 | `#002FA7`（克莱因蓝） |
| 文字色 | `#FFFFFF`（白色） |
| 字体 | Microsoft YaHei，11pt，**粗体** |
| 对齐 | 水平居中，垂直居中 |
| 边框 | 底部 `#E2E8F0` 1pt 实线 |

#### 数据行

| 属性 | 规范 |
|------|------|
| 背景色 | 白色（偶数行）/ `#F8FAFC`（奇数行） |
| 文字色 | `#1E293B` |
| 字体 | Microsoft YaHei，10pt，常规 |
| 数字格式 | 千位分隔符（如 `1,234`）、保留两位小数（金额类） |
| 边框 | 全部 `#E2E8F0` 0.5pt 实线 |
| 交替行 | 使用样式实现浅灰交替 |

#### 汇总行（末行）

| 属性 | 规范 |
|------|------|
| 背景色 | `#CAD5F6`（浅蓝紫） |
| 文字色 | `#002FA7`（克莱因蓝） |
| 字体 | Microsoft YaHei，10pt，**粗体** |
| 顶部边框 | `#002FA7` 2pt 双线 |

## 🎯 设计审查清单

生成 Word 文档后，必须核对：

- [ ] 标题 1/2/3 是否使用品牌色（`#002FA7` / `#00C3FF`）
- [ ] 是否有多余的颜色引入（红/橙/紫/粉等，RAG 除外）
- [ ] 表格表头是否为 `#002FA7` 背景 + 白色文字
- [ ] 图表配色是否使用品牌色板，而非 Word 默认配色
- [ ] 页眉页脚是否包含 Comlan 品牌标识
- [ ] 是否有封面页（如需要）
- [ ] 字体是否统一使用 Microsoft YaHei
- [ ] 数字格式是否统一（千位分隔、小数位数）
