# Scrum Artifacts Management

## 1. Product Backlog

User Story format:/n```
As a <role>
I want <feature>
So that <value>
```

Acceptance Criteria (Given-When-Then):
```
Given [precondition]
When [action]
Then [result]
```

Story Points: 1,2,3,5,8,13,21
- >=8 must be broken down

## 2. Sprint Backlog
Task format: [Task] action + expected result

## 3. Product Increment
Must meet Definition of Done.
