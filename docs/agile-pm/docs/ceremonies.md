# Scrum Ceremonies Guide

## 1. Sprint Planning
Goal: Plan Sprint work.
Duration: max 4h (2-week Sprint).

## 2. Daily Standup
Goal: Sync progress.
Duration: fixed 15 min.

Three questions:/n1. What did I do yesterday?
2. What will I do today?
3. Any blockers?

## 3. Sprint Review
Goal: Demo increment, get feedback.
Duration: max 2h.

## 4. Sprint Retrospective
Goal: Improve process.
Duration: max 1.5h.
