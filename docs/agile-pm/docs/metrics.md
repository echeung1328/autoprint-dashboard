# Metrics and Tracking

## Core Metrics

### 1. Velocity
Story points completed per sprint.
Use: Predict future capacity.

### 2. Sprint Burndown
Remaining work trend.
X-axis: sprint days (1-14)
Y-axis: remaining story points

### 3. Lead Time
Idea to production duration.
Goal: shorter is better.

### 4. Cycle Time
Development start to Done.
Goal: 2-5 days.

### 5. Defect Rate
Bugs found per sprint.
Goal: <=1 escaped bug/sprint.
