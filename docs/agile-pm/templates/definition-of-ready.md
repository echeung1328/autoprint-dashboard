# Definition of Ready

> Items not meeting DoR should not enter Sprint.

## DoR Checklist

### Content
- [ ] User story format complete
- [ ] Acceptance criteria clear
- [ ] Dependencies identified

### Estimable
- [ ] Story points estimated
- [ ] Points <= 8

### Testable
- [ ] Acceptance criteria convertible to tests

## DoR Grades

| Grade | Action |
|-------|--------|
| Ready | Enter Sprint |
| Nearly Ready | Refine this week |
| Not Ready | Back to backlog |
