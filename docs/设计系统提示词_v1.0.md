# AutoPrint 设计系统提示词 v1.0

> 适用场景：新项目启动时使用此提示词，让 AI 快速理解品牌规范与布局模式。
> 直接复制全文粘贴至对话框即可。

---

## 品牌色规范（强制）

本项目的品牌色仅使用以下两种颜色，禁止随意引入其他色相：

| 角色 | 色值 | 名称 | 用途 |
|------|-------|------|------|
| 主色 | `#002FA7` | 克莱因蓝 | 主标题、主按钮、KPI 卡片、激活状态 |
| 辅色 | `#00C3FF` | 浅蓝 | 次要强调、图表柱、耗时 KPI、hover 态 |
| 深色变体 | `#001D70` | 深克莱因蓝 | 最高月卡片、深色背景区域 |

**RAG 语义色（Red-Amber-Green，红-琥珀-绿，仅用于状态指示，不可用于装饰）：**

按 R→A→G 顺序排列，严格对应状态严重程度：

| 状态 | 色值 | 用途 |
|------|-------|------|
| 失败/异常/告警（Red） | `#ef4444` | 失败 badge、错误提示、异常状态、数值超标 |
| 警告/临界（Amber） | `#f59e0b` | 数值接近阈值、需要关注、待处理事项 |
| 成功/正常（Green） | `#10b981` | 成功 badge、正常状态指示、数值达标 |

> ⚠️ **设计纪律**：整体页面颜色不超过 3 种（品牌色 + 中性灰）。RAG 色仅用于状态指示，不用于装饰性元素。

## 布局模式规范

### 页面结构顺序

```
┌─────────────────────────────────┐
│  Header（标题 + 用户信息）       │
├─────────────────────────────────┤
│  筛选栏（日期 + 下拉 + 按钮）    │  ← 固定顶部
├─────────────────────────────────┤
│  Tab 导航（如有多视图）          │  ← 可选
├─────────────────────────────────┤
│  KPI 卡片区（4 列，响应式）     │
├─────────────────────────────────┤
│  负载高峰横幅（可选）            │
├─────────────────────────────────┤
│  图表区（2 列 → 1 列响应式）   │
├─────────────────────────────────┤
│  数据明细表格                    │
└─────────────────────────────────┘
```

### KPI 卡片规范

- **数量**：最多 4 张，横向排列，移动端自动堆叠
- **样式**：白色背景、12px 圆角、轻微阴影 `0 2px 8px rgba(0,0,0,0.08)`
- **顶部色条**：4px 高度的渐变色条（`#002FA7 → #00C3FF`）
- **数值字号**：32px，粗体 700
- **标签字号**：13px，颜色 `#64748b`

```css
.kpi-card {
  background: white;
  border-radius: 12px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.08);
  padding: 24px;
  position: relative;
}
.kpi-card::before {
  content: '';
  position: absolute;
  top: 0; left: 0; right: 0;
  height: 4px;
  border-radius: 12px 12px 0 0;
  background: linear-gradient(90deg, #002FA7, #00C3FF);
}
```

## 图表配色方案

### 柱状图 / 折线图

| 数据系列 | 颜色 |
|----------|-------|
| 主系列（总数/主要指标） | `#002FA7` |
| 次系列（成功/次要指标） | `#00C3FF` |
| 峰值高亮柱 | `#001D70`（深色变体） |
| 网格线 | `#e2e8f0` |
| 坐标轴文字 | `#64748b` |

### 饼图 / 环形图

使用品牌色渐变，禁止用彩虹色盘：

```javascript
const palette = ['#002FA7', '#001D70', '#00C3FF', '#66D9FF', '#93c5fd'];
```

## 组件样式规范

### 按钮

| 类型 | 背景 | 文字 | 边框 |
|------|-------|------|-------|
| 主按钮 | `#002FA7` | `#ffffff` | 无 |
| 次按钮 | 透明 | `#002FA7` | `1px solid #002FA7` |
| 危险按钮 | `#ef4444` | `#ffffff` | 无 |

### Badge（状态标签）

```css
.badge-success { background: #10b981; color: white; }
.badge-fail    { background: #ef4444; color: white; }
.badge-pending { background: #00C3FF; color: #002FA7; }
```

### 表格

- 奇数行：`#f8fafc`
- 悬停行：`#e2e8f0`
- 表头：粗体、`#1e293b`、底部 `2px solid #002FA7`

## Tab 导航规范（多视图页面必用）

当页面有多个独立视图时，**必须使用 Tab 切换**，禁止在同一页面堆叠多个内容区块。

```html
<div class="tab-nav">
  <button class="tab-btn active">📊 数据概览</button>
  <button class="tab-btn">📅 年度高峰</button>
</div>
<div class="tab-panel" id="panelOverview">...</div>
<div class="tab-panel" id="panelPeaks" style="display:none;">...</div>
```

```css
.tab-nav {
  display: flex;
  gap: 4px;
  padding: 4px;
  background: #e2e8f0;
  border-radius: 10px;
  margin: 0 32px 20px;
}
.tab-btn {
  flex: 1;
  padding: 10px 20px;
  border: none;
  border-radius: 8px;
  background: transparent;
  color: #64748b;
  font-weight: 500;
  cursor: pointer;
  transition: all 0.2s;
}
.tab-btn.active {
  background: white;
  color: #002FA7;
  box-shadow: 0 1px 4px rgba(0,0,0,0.1);
}
```

## 设计审查清单

每次完成页面开发，必须核对：

- [ ] 页面中是否只使用了品牌色 + 中性灰（RAG 色仅用于状态）
- [ ] 是否有多余的颜色引入（红/橙/紫/粉等）
- [ ] KPI 卡片是否统一使用顶部色条，而非左边框
- [ ] 图表配色是否使用品牌色板，而非默认彩虹色
- [ ] 多视图内容是否使用 Tab 切换隔离
- [ ] 移动端（< 768px）布局是否自动适配

**下次新项目开始时，将以下内容粘贴给 AI：**

```

- 主色：#002FA7（克莱因蓝）

【布局模式】
- 页面结构：Header → 筛选栏 →（可选）Tab 导航 → KPI 卡片 → 图表区 → 数据表格
- 多视图内容必须使用 Tab 切换，禁止堆叠显示

【KPI 卡片】
- 白色背景、12px 圆角、顶部 4px 品牌渐变色条
- 数值 32px 粗体，标签 13px 次要文字色

【图表配色】
- 主系列 #002FA7，次系列 #00C3FF
- 禁止使用彩虹色盘

请严格遵守以上规范，不要自行引入其他颜色。
```
